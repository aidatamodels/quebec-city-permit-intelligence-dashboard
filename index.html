À propos du dashboard

Objectif

Ce dashboard permet d’explorer les permis délivrés à la Ville de Québec, de visualiser les zones de concentration et d’analyser les tendances selon les filtres sélectionnés.

Comment utiliser la carte

  • Utilisez les filtres dans le panneau de gauche.
  • Déplacez-vous et zoomez sur la carte.
  • Survolez un point pour obtenir un aperçu rapide.
  • Cliquez sur un point pour ouvrir une fiche détaillée.

Approche de données

Les données affichées sont agrégées. Le dashboard évite d’exposer les adresses exactes et privilégie une représentation par zones afin de préserver une lecture plus responsable.

Modèles

Le projet utilise Python et Scikit-learn pour préparer les données, entraîner les modèles et générer le fichier dashboard_data.json utilisé par cette page.
